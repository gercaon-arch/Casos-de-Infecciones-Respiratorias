# Casos-de-Infecciones-Respiratorias
Solución de procesamiento batch para optimizar la vigilancia epidemiológica en Colombia usando Datos Abiertos (INS/SIVIGILA). Analiza contagios, muertes y brotes regionales mediante Big Data. Stack: Python y PySpark para procesamiento, Hadoop HDFS para almacenamiento masivo y Docker para el entorno. Transforma datos brutos en estrategia.
